SODAS+ 오픈 데이터 생태계 개요
==============================================================================

SODAS+ 는 오픈 데이터 생태계 구축 플랫폼으로, 생태계에 참여하여 데이터 공유를 지원하는 ``데이터 허브 (DataHub)`` 와
생태계의 관리 감독을 담당하는 ``거버넌스 시스템 (Governance System)`` 으로 구성된다.
